# Simple Book Shop Website

This is a simple responsive, multi page based website using React and React Router.

## Technology used

-   React, React Router
-   Tailwind
-   DaisyUI

## Webpage Sections:

1. Navbar section
2. Banner section
3. Fetch Data from local file
4. Show all data in grid view
5. Show single data using react router
6. FAQ section
7. Footer section
